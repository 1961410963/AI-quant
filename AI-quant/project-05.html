一、分类型机器学习算法

本项目使用三种经典分类算法进行对比实验：逻辑回归、决策树、随机森林

逻辑回归 线性分类

通过Sigmoid函数将线性输出转换为0~1概率值，用于二分类。可解释性强，训练速度快。

优点
  • 输出概率，便于设阈值
  • 训练快速
  • 可解释性好
缺点
  • 只能处理线性可分
  • 对特征尺度敏感
  • 需标准化

决策树 非线性

通过一系列if-else规则递归分裂样本，形成树状结构。可解释性强，能捕捉非线性关系。

优点
  • 可解释性强可可视化
  • 捕捉非线性关系
  • 不需标准化
缺点
  • 容易过拟合
  • 对数据变化敏感
  • 单棵树不稳定

随机森林 集成学习

多棵决策树通过Bagging集成，样本随机+特征随机，最后投票/平均输出。准确率高，泛化能力强。

优点
  • 准确率高
  • 泛化能力强
  • 能处理高维数据
缺点
  • 训练预测较慢
  • 模型占内存大
  • 可解释性差（黑盒）

二、模型评价指标

混淆矩阵、ROC曲线与AUC是分类模型最核心的评估工具

混淆矩阵示意

预测: 涨(1)
预测: 跌(0)
实际: 涨(1)
真正例
（实际涨·预测涨）
假反例
（实际涨·预测跌）
实际: 跌(0)
假正例
（实际跌·预测涨）
真反例
（实际跌·预测跌）

指标计算公式

准确率 = (真正例+真反例) / 总数

精确率 = 真正例 / (真正例+假正例)

召回率 = 真正例 / (真正例+假反例)

F1分数 = 2 × 精确率 × 召回率 / (精确率+召回率)

真阳性率 = 真正例 / (真正例+假反例)（纵轴）

假阳性率 = 假正例 / (假正例+真反例)（横轴）

💡 AUC 解读： AUC = 1 完美分类 · AUC > 0.8 模型良好 · AUC > 0.7 模型有效 · AUC = 0.5 随机猜测（无效）
