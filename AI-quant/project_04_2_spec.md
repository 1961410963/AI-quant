# 项目04.2：海龟策略演示看板 Spec文档

## 1. 项目概述

### 1.1 项目目标
开发一个交互式海龟策略演示看板，支持用户选择不同标的、时段和策略参数，实时回测并展示结果，帮助用户理解海龟法则的适应场景和使用心得。

### 1.2 核心功能
| 功能模块 | 描述 |
|---------|------|
| 标的选择 | 支持从已存储数据中选择不同股票/ETF标的 |
| 时段选择 | 支持3年/1年/6个月/3个月等不同回测周期 |
| 参数配置 | 支持调节入场通道周期、退出通道周期、ATR周期、加仓步长、最大单位数等核心参数 |
| 实时回测 | 根据配置参数实时计算回测指标和交易结果 |
| 可视化展示 | 净值曲线、交易信号图、ATR走势图、交易明细表 |
| 数据更新 | 支持一键补充最新数据 |

### 1.3 技术栈
- **前端**：HTML5 + JavaScript + ECharts 5.x
- **后端**：Python Flask（可选）或纯前端解析CSV
- **数据存储**：CSV文件（前复权格式）
- **图表库**：ECharts

## 2. 功能模块设计

### 2.1 左侧控制面板

#### 2.1.1 标的选择
- 下拉菜单展示所有可用标的
- 自动读取 `output/` 目录下的 `*_daily_data.csv` 文件
- 显示标的代码和名称

#### 2.1.2 时段选择
- 按钮组：3年、1年、6个月、3个月
- 默认选中3年

#### 2.1.3 策略参数配置
| 参数名称 | 默认值 | 范围 | 说明 |
|---------|--------|------|------|
| 入场通道周期(N) | 20 | 10-100 | 唐奇安通道入场周期 |
| 退出通道周期(M) | 10 | 5-50 | 反向突破离场周期 |
| ATR周期 | 14 | 5-30 | 平均真实波幅计算周期 |
| 加仓步长(xATR) | 0.5 | 0.1-2.0 | 每上涨多少ATR加仓 |
| 最大单位数 | 4 | 1-10 | 最多持有单位数 |
| 止损倍数 | 2 | 1-5 | 止损距离倍数 |
| 单单位风险(%) | 1 | 0.1-5 | 每单位头寸风险占比 |
| 初始资金(元) | 100000 | 10000-10000000 | 回测初始资金 |

- 使用滑块组件调节参数
- 实时显示当前参数值

#### 2.1.4 操作按钮
- **回测**：点击后执行回测并更新图表
- **重置参数**：恢复默认参数
- **更新数据**：调用Python脚本获取最新数据

### 2.2 右侧展示面板

#### 2.2.1 回测指标卡片
| 指标 | 格式 | 颜色规则 |
|------|------|----------|
| 年化收益 | 百分比 | >0绿色，<0红色 |
| 夏普比率 | 保留2位小数 | >1绿色，0-1黄色，<0红色 |
| 最大回撤 | 百分比 | <10%绿色，10-20%黄色，>20%红色 |
| 胜率 | 百分比 | >50%绿色，<50%红色 |
| 交易笔数 | 整数 | 灰色 |
| 止损次数 | 整数 | 灰色 |

#### 2.2.2 净值曲线图
- 展示策略净值曲线
- 对比买入持有曲线
- 支持缩放和平移
- 显示累计回报、年化回报、超额收益

#### 2.2.3 交易信号图
- 展示股价、唐奇安通道（上轨/下轨）
- 标记买入信号（绿色三角）和卖出信号（红色三角）
- 标记加仓点位（蓝色三角）

#### 2.2.4 ATR走势图
- 柱状图展示ATR变化
- 显示当前ATR值

#### 2.2.5 交易明细表
- 展示所有交易记录
- 包含：日期、类型、价格、数量、金额、佣金、盈亏、退出原因
- 支持滚动查看

## 3. 数据结构设计

### 3.1 CSV数据格式
```csv
trade_date,open,high,low,close,vol,amount
2023-07-10,18.54,18.57,18.08,18.24,41454,76323326.0
```

### 3.2 指标计算逻辑

#### 3.2.1 唐奇安通道
```
entry_high = high.rolling(N).max()
entry_low = low.rolling(N).min()
exit_low = low.rolling(M).min()
```

#### 3.2.2 ATR计算
```
TR = max(high-low, |high-prev_close|, |low-prev_close|)
ATR = TR.rolling(atr_period).mean()
```

#### 3.2.3 交易信号
```
buy_signal = close > prev(entry_high)
sell_signal = close < prev(exit_low)
```

#### 3.2.4 仓位计算
```
shares_per_unit = (capital * risk_per_unit) / ATR
```

#### 3.2.5 加仓规则
```
add_on_price = entry_price + current_units * add_on_atr * ATR
if high >= add_on_price and current_units < max_units:
    add_position(shares_per_unit)
```

#### 3.2.6 止损规则
```
stop_loss_price = entry_price - stop_loss_multiplier * ATR
if low <= stop_loss_price:
    sell(stop_loss_price)
```

## 4. 回测指标计算公式

### 4.1 累计回报
```
累计回报 = (期末资产 - 期初资产) / 期初资产 × 100%
```

### 4.2 年化回报
```
年化回报 = (期末资产 / 期初资产)^(365/N) - 1 × 100%
N = 回测自然日天数
```

### 4.3 最大回撤(MDD)
```
MDD = min_t((当日资产 - 历史最高资产) / 历史最高资产) × 100%
```

### 4.4 夏普比率
```
夏普比率 = √252 × (日均超额收益 / 日均收益标准差)
超额收益 = 策略收益 - 无风险利率(2%)
```

### 4.5 胜率
```
胜率 = 盈利交易次数 / 总交易次数 × 100%
```

### 4.6 盈亏比
```
盈亏比 = 平均盈利 / |平均亏损|
```
