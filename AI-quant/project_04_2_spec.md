## 1. 项目概述

### 1.1 项目目标
开发一个**交互式海龟策略演示看板**，帮助用户通过调节策略参数观察收益变化，理解海龟法则的适应场景和使用心得。

### 1.2 设计理念
- **差异化设计**：顶部控制条+全宽卡片流布局（无侧边栏），与老师的工具视觉风格不同
- **原创模块**：策略能力雷达图（六维评估）、参数预设芯片、心得总结卡片
- **交互友好**：参数数值+隐藏滑块+进度条三联动，可直接输入精确值

### 1.3 核心功能
| 功能模块 | 描述 |
|---------|------|
| 标的选择 | 8只标的（天邑股份、恩捷股份、紫金矿业、比亚迪、宁德时代、贵州茅台、中国平安、上证50ETF） |
| 时段选择 | 3年/1年/6月/3月按钮组 |
| 参数预设 | 4种芯片式预设（标准海龟/稳健保守/激进进攻/短线波段） |
| 参数配置 | 8个可调节参数，数值+滑块+进度条三联动 |
| 实时回测 | 点击"回测"按钮即时计算，加载动画提示 |
| 指标展示 | 6个左侧色条指标卡（颜色随盈亏自动变化） |
| 可视化 | 净值曲线+回撤叠加、唐奇安通道信号图、策略能力雷达图 |
| 交易明细 | 完整交易记录表（日期/类型/价格/数量/金额/佣金/盈亏/原因） |
| 心得总结 | 6张色卡展示海龟法则适应场景与使用心得 |

## 5. 策略计算逻辑

### 5.1 ATR计算（Wilder平滑法）
```javascript
// TR计算
TR = MAX(high - low, |high - prev_close|, |low - prev_close|)

// Wilder平滑法（初始SMA种子后递归）
ATR_initial = SUM(TR[0:N]) / N
ATR_t = (ATR_{t-1} × (N-1) + TR_t) / N
```

### 5.2 唐奇安通道
```javascript
entry_high[i] = MAX(high[i-N+1 : i+1])
exit_low[i] = MIN(low[i-M+1 : i+1])
```

### 5.3 交易信号
```javascript
buy_signal[i] = close[i] > entry_high[i-1]
sell_signal[i] = close[i] < exit_low[i-1]
```

### 5.4 仓位计算
```javascript
shares_per_unit = FLOOR(capital × risk% / ATR / 100) × 100
```

### 5.5 加仓规则
```javascript
add_on_price = entry_price + current_units × addon × ATR
if (high >= add_on_price && units < max_units) {
    add_position(shares_per_unit)
    units++
}
```

### 5.6 止损规则
```javascript
stop_loss_price = entry_price - stop_multiplier × ATR
if (low <= stop_loss_price) {
    sell_at(stop_loss_price)
}
```

## 6. 回测执行细节

### 6.1 滑点处理
| 操作 | 滑点公式 | 说明 |
|------|---------|------|
| 买入 | `open × 1.0001` | +0.01%冲击成本 |
| 止损卖出 | `stop_price × 0.9999` | -0.01%冲击成本 |
| 止盈卖出 | `open × 0.9999` | -0.01%冲击成本 |
| 加仓 | `trigger_price × 1.0001` | +0.01%冲击成本 |

### 6.2 手续费
| 费项 | 费率 | 计算方式 |
|------|------|---------|
| 买入佣金 | 0.03% | 成交金额 × 0.0003 |
| 卖出佣金 | 0.03% | 成交金额 × 0.0003 |
| 印花税 | 0.05% | 卖出金额 × 0.0005（仅卖出） |

### 6.3 回测指标计算
```javascript
// 累计回报
cumRet = (finalAsset - initialCapital) / initialCapital × 100

// 年化回报
annRet = (finalAsset / initialCapital)^(365/tradingDays) - 1 × 100

// 最大回撤
for each day: mdd = MIN((asset - maxAsset) / maxAsset × 100)

// 夏普比率
dailyReturnMean = MEAN(dailyReturns)
dailyReturnStd = STD(dailyReturns)
sharpe = SQRT(252) × (dailyReturnMean - 0.02/252) / dailyReturnStd

// 胜率
winRate = winCount / totalTradeCount × 100

// 盈亏比
avgWin = totalProfit / winCount
avgLoss = totalLoss / loseCount
plRatio = avgWin / avgLoss
```
