2. 核心概念详解

【海龟策略（Turtle Trading）】
海龟策略是由传奇交易员理查德·丹尼斯（Richard Dennis）和威廉·埃克哈特（William Eckhardt）在1983年创立的系统化交易策略。其核心思想是"顺势而为"——在市场突破时入场，跟随趋势获利，通过严格的止损和仓位管理控制风险。策略名称来源于丹尼斯的名言："我要培养交易员，就像新加坡人养海龟一样容易。"
【高低点通道（Donchian Channel）】
高低点通道，又称唐奇安通道，是海龟策略的核心入场信号指标。它由三条线组成：
上轨（High Channel）：过去N个交易日的最高价
下轨（Low Channel）：过去N个交易日的最低价
中轨（Mid Channel）：上轨与下轨的平均值
当价格突破上轨时产生买入信号，突破下轨时产生卖出信号。
【平均真实波幅（ATR - Average True Range）】
ATR是衡量市场波动性的指标，由威尔斯·怀尔德（J. Welles Wilder Jr.）发明。真实波幅（TR）取以下三者中的最大值：
1. 当日最高价 - 当日最低价
2. |当日最高价 - 前一日收盘价|
3. |当日最低价 - 前一日收盘价|
ATR是TR的N日Wilder平滑值（标准周期为14日）。ATR越大说明市场波动越剧烈，ATR越小说明市场越平稳。
【止损条件（Stop Loss）】
海龟策略使用ATR作为止损基准：
止损价 = 入场价 - 2×ATR
当价格跌破止损价时，立即卖出平仓，控制单笔交易的最大亏损。止损距离不是固定百分比，而是根据市场波动性动态调整——波动大时止损距离宽，波动小时止损距离窄。
【仓位管理（Unit）】
海龟策略最精妙的部分是仓位管理系统，核心概念是"单位"（Unit）：
可买股数 = (账户资金 × 1.0%) / ATR
一个单位的头寸，如果价格波动1个ATR，账户损失1.0%。这种方法确保了在不同波动性的市场中，每单位头寸承担相同的风险。
【金字塔加仓（Pyramid Adding）】
海龟策略在盈利时采用金字塔式加仓：首次入场建立1个单位，之后每上涨0.5ATR再加仓1个单位，最多持有4个单位。核心逻辑是"让盈利头寸贡献更多收益"，同时控制总风险（最多4个单位 = 4.0%风险）。

3. 双系统突破入场机制

系统一（S1）— 20日突破 / 10日离场

  • 入场条件：突破20日最高价 → 买入
  • 离场条件：跌破10日最低价 → 卖出
  • 特点：更敏感，更快捕捉趋势启动
  • 优势：交易信号更多，入场时机更早
  • 劣势：假突破多，容易受到市场噪音干扰

系统二（S2）— 55日突破 / 20日离场

  • 入场条件：突破55日最高价 → 买入
  • 离场条件：跌破20日最低价 → 卖出
  • 特点：更稳健，过滤更多噪音
  • 优势：趋势更确认，大趋势捕捉能力更强
  • 劣势：交易信号更少，入场时机稍晚

海龟们通常同时运行两个系统，分散风险，提高策略稳定性

4. 金字塔加仓规则

盈利时金字塔加仓（最多4个单位）

  • 首次入场：建立1个单位，止损价 = 入场价 - 2×ATR
  • 价格上涨0.5ATR：加仓1个单位，止损价保持不变
  • 再上涨0.5ATR：再加仓1个单位，累计3个单位
  • 再上涨0.5ATR：再加仓1个单位，累计4个单位（最大）
加仓核心逻辑：
- 在盈利时加仓，而非亏损时摊平成本（顺势而为）
- 让盈利头寸贡献更多收益（趋势越强，收益越大）
- 控制总风险（最多4个单位 = 4.0%账户资金风险）
- 禁止在亏损时加仓！不摊平成本，不放大亏损

5. 策略规则与参数

海龟策略通过突破高低点通道来识别趋势方向，结合ATR进行动态风险控制和仓位管理。
入场通道计算：
entry_high_S1 = high.rolling(20).max()（前20日最高价）
entry_high_S2 = high.rolling(55).max()（前55日最高价）
离场通道计算：
exit_low_S1 = low.rolling(10).min()（前10日最低价）
exit_low_S2 = low.rolling(20).min()（前20日最低价）
ATR计算（Wilder平滑法）：
TR = max(high-low, |high-prev_close|, |low-prev_close|)
ATR_t = (ATR_{{t-1}} × (14-1) + TR_t) / 14（前14个TR的加权平滑值）
买入规则（双系统突破）：
1. 系统一：收盘价突破前一日20日上轨时产生买入信号
2. 系统二：收盘价突破前一日55日上轨时产生买入信号
3. 任一系统触发即买入，次日开盘执行
4. 计算止损价 = 买入价 - 2×ATR
5. 计算仓位 = (账户资金 × 1.0%) / ATR（取整到100股），建立1个单位
加仓规则（金字塔）：
1. 持仓期间，若当前单位数 < 4，且价格上涨到入场价 + 当前单位数 × 0.5 × ATR
2. 以该价格加仓1个单位
3. 止损价保持不变（不扩大风险）
4. 最多加仓到4个单位
卖出规则（双重退出机制）：
1. 止损卖出：当日最低价跌破止损价时，以止损价卖出
2. 止盈卖出：收盘价跌破对应系统的离场通道下轨时，次日开盘卖出
- 系统一入场 → 跌破10日低点离场
- 系统二入场 → 跌破20日低点离场
3. 优先级：止损 > 止盈信号
4. 离场逻辑体现"让利润奔跑"理念，不轻易卖出
移动止损调整：
在持仓期间，若ATR下降，止损价会上移（锁定更多利润）；若ATR上升，止损价保持不变（不扩大风险）。
风险控制规则：
1. 单个市场最多4个单位，最大风险4.0%
2. 同一方向所有头寸合计不超过12个单位
3. 高度相关市场合计不超过6个单位
4. 账户总风险不超过12%
5. 触发止损必须立即执行，不能犹豫

10. 策略评估指标详解

【最大回撤（Maximum Drawdown, MDD）】从资产峰值到谷底的最大跌幅，衡量策略承受的最大亏损。
公式：MDD = min_t((当日资产 - 历史最高资产) / 历史最高资产) × 100%
本策略最大回撤：-25.35%，评级：风险较高（橙色）。
【夏普比率（Sharpe Ratio）】衡量每承担一单位风险所获得的超额收益。
公式：S = √252 × (日均超额收益 / 日均收益标准差)，其中超额收益 = 策略收益 - 无风险利率(2%)
本策略夏普比率：0.12（>2优秀，1-2良好，0-1一般，<0策略无效）。
【累计回报（Cumulative Return）】整个回测期的总收益。
公式：累计回报 = (期末资产 - 期初资产) / 期初资产 × 100%
本策略累计回报：4.73%，初始资金 100000 元最终变为 104726.26 元。
【年化收益率（Annualized Return）】将收益标准化到每年。
公式：年化收益率 = (期末资产 / 期初资产)^(365/N) - 1（N为回测自然日天数）
本策略年化回报：1.55%，评级：一般。
【超额收益（Excess Return）】策略收益与买入持有收益的差异。
公式：超额收益 = 策略收益率 - 买入持有收益率
本策略超额收益：12.29%，买入持有收益：-7.57%。
【胜率（Win Rate）】盈利交易次数占总交易次数的比例。
公式：胜率 = 盈利交易次数 / 总交易次数 × 100%
本策略胜率：42.86%。
【盈亏比（Profit/Loss Ratio）】平均盈利与平均亏损的倍数。
公式：盈亏比 = 平均盈利 / |平均亏损|
本策略盈亏比：1.57。
【期望收益（Expected Return）】每次交易的期望结果。
公式：期望收益 = 胜率 × 平均盈利 - (1-胜率) × 平均亏损
本策略期望收益：359.91元/次交易。

11. 总结分析

11.1 核心结论

一句话结论：海龟策略在天邑股份上3年累计回报4.73%，与买入持有（-7.57%）相比超额收益12.29%，策略表现一般。

策略 vs 持有：策略累计回报4.73%，买入持有-7.57%，超额收益12.29%。28笔交易中仅6笔盈利（胜率42.86%），盈亏比1.57，每次交易期望359.91元。

最大回撤：策略最大回撤-25.35%，ATR动态止损、金字塔加仓和单位风险控制机制有效控制了风险。

11.2 收获与反思

收获一：理解了海龟策略的五大核心要素。完整的交易系统必须包含市场选择、头寸规模、入场规则、止损规则和离场规则五个要素，缺一不可。海龟策略的精髓在于将这些要素系统化、规则化，避免情绪化操作。

收获二：体验了双系统突破的优势。同时运行系统一（20日/10日）和系统二（55日/20日）可以分散风险。系统一更敏感，能快速捕捉趋势启动；系统二更稳健，过滤噪音能力更强。两者结合提高了策略的稳定性。

收获三：掌握了基于ATR的仓位管理方法。通过"单位"概念，每单位头寸承担相同的风险（1.0%账户资金），确保在不同波动性市场中风险一致。这是海龟策略最精妙的设计之一。

收获四：认识到金字塔加仓的价值。在盈利时加仓而非亏损时摊平成本，符合"顺势而为"理念。最多4个单位的限制确保了总风险可控（4.0%）。

收获五：理解了"让利润奔跑"的离场哲学。反向突破离场而非固定止盈，能够捕捉完整趋势。系统一用10日低点、系统二用20日低点作为离场信号，既保护利润又不提前出局。

反思一：通道周期的选择。标准20日/55日入场周期和10日/20日离场周期在天邑股份上的表现取决于其趋势特征。如果标的长期处于震荡格局，较短周期的通道可能产生过多假信号。

反思二：单单位风险的设定。1.0%的单单位风险是标准设定，但可以根据个人风险承受能力调整。风险偏好低的投资者可以降低此比例。

反思三：缺乏趋势过滤。策略在所有市场状态下都执行相同逻辑。如果能先判断市场是否处于趋势状态（如用ADX指标），在震荡市中暂停交易，可以减少假信号亏损。

11.3 后续优化计划

优化方向一：引入趋势过滤指标。在突破信号基础上增加ADX指标过滤。当ADX<25（无明显趋势）时不产生交易信号，避免在震荡市中频繁操作。

优化方向二：调整通道周期参数。尝试不同的入场/离场通道周期组合，找到最适合天邑股份的参数组合。

优化方向三：优化加仓策略。可以考虑根据趋势强度动态调整加仓间隔，趋势越强加仓越频繁。

优化方向四：引入移动止盈。当价格上涨一定幅度后，动态调整止损线为移动止损，锁定更多利润。

优化方向五：结合基本面分析。在技术面信号基础上，结合公司基本面进行过滤，避免在基本面恶化的标的上交易。

11.4 适用场景总结

适合海龟策略的场景：
1. 处于明显上升或下降趋势的标的
2. 价格波动较大、趋势持续时间较长的标的
3. 日均成交额大于5亿元、流动性好的标的
4. 不受重大事件驱动、技术规律较强的标的

不适合海龟策略的场景：
1. 长期横盘整理的震荡市标的
2. 日均成交额小于2亿元、流动性不足的小盘股
3. 受政策或事件驱动、波动剧烈且无规律的标的
4. 基本面恶化、存在退市风险的标的

天邑股份定位：基于近三年数据，天邑股份的趋势特征决定了海龟策略的适用性。需要结合实际回测结果判断是否适合该策略。
