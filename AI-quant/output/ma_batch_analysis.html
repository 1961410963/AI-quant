横轴为4组均线周期，左纵轴为正收益占比(%)（该周期下盈利标的数/总标的数），右纵轴为平均夏普比率。正收益占比反映参数组合的"普适性"——越高说明该参数在越多标的上有效。

三、核心指标计算逻辑

1. 累计回报：回测期末总资产与初始资金的差额比例。
公式：累计回报 = (期末总资产 - 初始资金) / 初始资金 × 100%
例如：初始10万，期末9.5万，累计回报 = (9.5-10)/10 = -5%。
2. 买入持有收益（对照基准）：假设回测首日将全部资金一次性买入标的，之后不做任何交易、一直持有到期末的收益。
公式：买入持有收益 = (期末收盘价 - 首日收盘价) / 首日收盘价 × 100%
用途：作为策略的对照基准，衡量"什么都不做"的收益水平。数据已使用前复权，分红送转已自动调整。
3. 超额收益：策略收益与买入持有收益的差额，衡量策略是否创造了额外价值。
公式：超额收益 = 累计回报 - 买入持有收益
重要提醒：超额收益为正 ≠ 策略赚钱。例如策略亏-5%、持有亏-10%，超额 = (-5%) - (-10%) = +5%。这表示"虽然亏了，但比傻拿着少亏5%"，策略跑赢了基准，但绝对收益仍是负的。超额为负则表示策略连"什么都不做"都不如。
4. 区间最大回撤：整个回测期间，从资产峰值到谷底的最大跌幅。
计算步骤：① 每日计算当前回撤 = (当日总资产 - 历史最高总资产) / 历史最高总资产 × 100%；② 取所有日回撤中的最小值（最负值）。
例如：资产从10万涨到11万（峰值），再跌到9万，区间最大回撤 = (9-11)/11 = -18.18%。
5. 夏普比率：衡量每承担一单位风险所获得的超额收益（相对无风险利率）。
公式：Sharpe = (年化收益率 - 无风险利率) / 年化波动率
无风险利率取2%（近似一年期存款利率）。夏普 > 0 说明策略跑赢无风险理财，< 0 说明不如存银行。
6. 胜率与盈亏比：
胜率 = 盈利交易次数 / 总交易次数 × 100%
盈亏比 = 平均盈利金额 / 平均亏损金额
趋势策略通常胜率低（30%-40%）但盈亏比高（>1）——亏的时候小亏（死叉止损），赚的时候大赚（趋势延续）。若盈亏比 < 1，说明每次盈利覆盖不了亏损，策略在震荡市中失效。

四、策略适用场景与应用心得

1. 适用场景：双均线策略本质是趋势跟踪策略，其盈利前提是标的走出持续的单边趋势。从实验结果看，趋势性强的标的（如黄金ETF等）更适合，因为价格能持续创新高/新低，金叉后能持续运行；而震荡市中双均线策略会反复触发假信号，导致高胜率亏损——金叉买入后立即死叉卖出，来回吃佣金和滑点。
2. 均线周期选择：短周期（MA5/MA10）反应灵敏但信号频繁，适合波动大、趋势短的标的，但交易成本侵蚀严重；长周期（MA20/MA60）信号少、滞后大，能过滤噪音但会错过趋势启动初期的利润。经典组合MA5/MA20是平衡点。周期选择应匹配标的的波动节奏：慢牛/慢熊用长周期，活跃标的用中短周期。
3. 胜率与盈亏比的反向关系：双均线策略通常胜率偏低（本次实验普遍在30%-40%）但盈亏比应较高——因为每次亏损被严格限制在死叉时的小幅回撤，而盈利时趋势能跑出大段利润。若发现胜率低且盈亏比也低于1，说明标的处于震荡市，策略失效。本次实验中部分组合盈亏比偏低，反映出近三年市场震荡特征明显。
4. 全仓策略的风险特征：本次采用全仓买入/全仓卖出，金叉时All in，死叉时清仓。这种方式在趋势明确时收益最大化，但在震荡市中连续假信号会导致较大回撤——金叉买入后马上死叉卖出，每次都要承担佣金+滑点+印花税成本。全仓策略对择时要求极高，一旦趋势判断错误，单次亏损幅度远大于轻仓策略。建议在实盘前先确认标的处于趋势市而非震荡市。
5. 应用心得总结：① 双均线策略不是"万能公式"，必须先判断市场状态（趋势/震荡）再决定是否启用；② 择时不如择势——先选趋势性强的标的，再谈参数优化；③ 参数不是越精细越好，过度拟合历史数据反而降低未来适应性；④ 止损纪律比信号本身更重要，死叉即走，不扛单；⑤ 建议将双均线作为趋势确认工具而非唯一信号，结合成交量、MACD等指标过滤假信号效果更佳。
