1. 策略规则与参数

双均线策略是经典的趋势跟踪策略，通过短期均线MA5与长期均线MA20的交叉信号来判断市场趋势变化。
信号编码：每日收盘后计算状态信号，非交易指令
MA5 = close.rolling(5).mean()（近5日收盘价均值）
MA20 = close.rolling(20).mean()（近20日收盘价均值）
signal = 1（多头排列，MA5>MA20） / -1（空头排列，MA5<MA20） / 0（持平）
交叉捕捉：使用差值捕捉均线穿越瞬间
cross_signal[t] = signal[t-1] - signal[t-2]
cross_signal[t] = 2 → 第t-1日收盘时金叉（MA5上穿MA20），第t日开盘买入
cross_signal[t] = -2 → 第t-1日收盘时死叉（MA5下穿MA20），第t日开盘卖出
买入规则（金叉触发）：
1. 可用资金 = 当前现金 × 10%（约1万元）
2. 买入价格 = 第t日开盘价 × (1 + 0.0001)（含滑点万1）
3. 买入数量 = floor(可用资金 / 买入价格 / 100) × 100（取整到100股）
4. 交易成本 = 买入金额 × 佣金率(0.0003)
5. 现金减少 = 买入金额 + 佣金
6. 持仓增加 = 买入数量
卖出规则（死叉触发）：
1. 卖出价格 = 第t日开盘价 × (1 - 0.0001)（含滑点万1）
2. 卖出数量 = 当前全部持仓（清仓）
3. 交易成本 = 卖出金额 × 佣金率(0.0003) + 卖出金额 × 印花税(0.0005)
4. 现金增加 = 卖出金额 - 佣金 - 印花税
5. 持仓清零
6. 盈亏 = 净收入 - 持仓成本（买入价 × 数量）
避免未来函数：第 t 日决策只用 t-1 日及之前的信息
用第t-1日收盘计算的MA5/MA20判断交叉，在第t日开盘执行交易
完整策略思路：
第一步：数据准备 — 获取天邑股份前复权日线数据（开盘价、收盘价、成交量），截取近三年数据。
第二步：指标计算 — 每日收盘后计算MA5（5日均线）和MA20（20日均线），生成多空头状态信号。
第三步：信号识别 — 通过signal.diff()捕捉均线穿越瞬间：cross_signal=2表示金叉（买入信号），cross_signal=-2表示死叉（卖出信号）。
第四步：交易执行 — 收到信号后，次日开盘执行交易。买入时用10%现金（约1万元），按开盘价×1.0001（含滑点）计算可买数量，取整到100股；卖出时清空全部持仓，按开盘价×0.9999（含滑点）计算成交价。
第五步：成本扣除 — 买入扣万三佣金，卖出扣万三佣金+万五印花税。
第六步：资产更新 — 每日收盘后计算总资产（现金+持仓市值），记录净值曲线和回撤曲线（每日当前回撤，即从当日净值到历史最高点的跌幅）。
第七步：指标统计 — 回测结束后计算累计回报、年化回报、夏普比率、胜率、盈亏比等评估指标。

5. 策略评估指标详解

【总收益率（Total Return）】衡量整个回测期的累计收益。
公式：总收益率 = (期末净值 - 期初净值) / 期初净值 × 100%
本策略累计回报：-5.36%，初始资金 100000 元最终变为 94639.43 元。
【年化收益率（Annualized Return）】将收益标准化到每年，便于跨期与跨策略比较。
公式：年化收益率 = (期末净值 / 期初净值)^(365/N) - 1（N为回测自然日天数）
本策略年化回报：-1.82%，评级：亏损（年化>10%优秀，5%-10%良好，0%-5%一般，<0%亏损）。
【超额收益（Excess Return）】策略收益与简单买入持有相比的差异。
公式：超额收益 = 策略收益率 - 买入持有收益率
本策略超额收益：2.21%，买入持有收益：HOLDING_-5.36%。若为正，说明策略创造了额外价值。
【回撤曲线】每日当前回撤，即当日资产净值距离历史最高点的跌幅。资产创新高时回撤为0，资产下跌时回撤为负，资产反弹时回撤向0收窄。它反映策略在不同时间点的"浮亏状态"，而非最终亏损。
【最大回撤（Maximum Drawdown）】从资产峰值到谷底的最大跌幅，衡量策略"最坏情况"，是回撤曲线中的最低点。
公式：MDD = max_t(峰值 - 谷底) / 峰值 × 100%
本策略最大回撤：-6.87%，评级：风险很低（绿色）（回撤<10%风险很低，10%-20%风险适中，20%-30%风险较高，>30%风险很高）。
【夏普比率（Sharpe Ratio）】衡量每承担一单位风险所获得的超额收益。
公式：S = (Rp - Rf) / σp（Rp为年化收益率，Rf为无风险利率2%，σp为年化波动率）
本策略夏普比率：-0.90（>2优秀，1-2良好，0-1一般，<0策略无效）。
【胜率（Win Rate）】盈利交易次数占总交易次数的比例。
公式：胜率 = 盈利交易次数 / 总交易次数 × 100%
本策略胜率：28.57%。高胜率不等于好策略，必须与盈亏比结合看。
【盈亏比（Profit/Loss Ratio）】平均盈利与平均亏损的倍数。
公式：盈亏比 = 平均盈利 / |平均亏损|
本策略盈亏比：0.96（>2意味着每次盈利可覆盖两次以上亏损）。
【期望收益（Expected Return）】每次交易的期望结果。
公式：期望收益 = 胜率 × 平均盈利 - (1-胜率) × 平均亏损
本策略期望收益：-251.70元/次交易。
